# 🍳 푸드하우스 운영 및 수익화 스킬 가이드 (비전공자용)

이 가이드는 비전공자이신 사용자님이 이 앱을 통해 실제로 광고 수입을 올리고, 카카오 로그인 및 모바일 앱으로 확장하는 노하우를 집약한 **스킬북**입니다.

---

## 1. 🌐 무료로 인터넷에 배포하기 (GitHub Pages)
개발 서버를 돈 주고 빌릴 필요 없이, GitHub을 통해 **100% 무료**로 전 세계에 이 앱을 공개할 수 있습니다.

1. **GitHub 가입 및 저장소 생성**
   - [GitHub 공식 사이트](https://github.com)에 가입합니다.
   - `New repository`를 눌러 저장소를 만듭니다. (예: 저장소 이름 `food`)
2. **파일 업로드**
   - 현재 폴더(`d:/food`)에 있는 파일들(`index.html`, `style.css`, `recipe_app_skills.md`)을 드래그 앤 드롭으로 업로드합니다.
3. **무료 웹사이트 활성화**
   - 저장소의 **Settings** -> **Pages** 메뉴로 이동합니다.
   - Build and deployment의 Source를 `Deploy from a branch`로 설정하고 Branch를 `main` (또는 `master`)으로 선택한 뒤 **Save**를 누릅니다.
   - 1~2분 뒤 `https://내아이디.github.io/food` 주소로 전 세계 누구나 접속할 수 있는 앱이 활성화됩니다.

---

## 2. 💵 광고 수익 붙이기 (구글 애드센스 & 카카오 애드핏)
운영자인 사용자님이 돈을 버는 첫 번째 기본 비즈니스 모델입니다.

1. **광고 플랫폼 신청**
   - 구글 애드센스(Google AdSense)나 카카오 애드핏(Adfit)에 가입 신청합니다.
   - 깃허브 페이지로 만든 여러분의 웹사이트 주소(`https://내아이디.github.io/food`)를 등록하고 승인을 받습니다.
2. **광고 코드 삽입**
   - 승인이 완료되면 광고 제공 업체로부터 `ins` 태그나 `<script>` 형태의 코드를 발급받습니다.
   - `index.html` 파일의 `AdBanner` 컴포넌트 내부(또는 상단)에 해당 코드를 붙여넣으면 즉시 실제 광고가 노출되며 클릭 시 수익이 적립됩니다.

## 3. 🎫 플랫폼 중개 수수료 수익 모델 (운영자 핵심 수입원)
광고 수익은 트래픽이 엄청나게 많아야 유의미한 돈이 되지만, **'맛집 할인 식사권 판매'**는 결제가 발생할 때마다 **고액의 수수료(예: 5%)**를 확실하게 가져올 수 있어 가장 빠르게 고수익을 낼 수 있는 모델입니다.

### 💡 작동 원리 (크리에이터 이코노미)
1. **작성자(유저)**: 맛집이나 카페 사진을 예쁘게 찍어 올리면서 해당 가게의 "할인 식사권"을 등록합니다.
2. **구매자(유저)**: 다른 유저의 맛집 리뷰를 보고, 마음에 들면 앱 내에서 "할인 식사권 구매하기"를 눌러 모바일 쿠폰을 저렴하게 삽니다.
3. **운영자(사용자님)**: 결제 금액의 **5%를 중개 수수료**로 가져갑니다.
4. **리워드 배분**: 구매를 유도한 작성자에게는 수수료의 일부(예: 2%)를 **포인트**로 지급해 줌으로써 자발적으로 맛집 리뷰를 많이 쓰게 만듭니다.

## 4. 👥 팔로우 및 크리에이터 이코노미 극대화 전략 (신규!)
커뮤니티 서비스의 핵심은 **"유저들이 매일 다시 방문하는 것(잔존율)"**입니다.

1. **팔로우 피드 작동**
   - 유저들이 마음에 드는 크리에이터를 팔로우하면, 홈 화면 가장 앞에 있는 **`팔로잉`** 탭에 그들의 최신 글만 정렬되어 노출됩니다.
   - 이를 통해 유저는 자신의 요리 취향에 맞는 피드만 엄선해 볼 수 있어 앱의 충성 고객이 됩니다.
2. **게시글 및 팔로워 통계**
   - 크리에이터 프로필에 게시글 수와 팔로워 수가 실시간 업데이트되어, 더 많은 팔로워를 모으고 싶은 욕구(인플루언서 욕구)를 자극합니다.
   - 이는 더 많은 양질의 맛집 리뷰와 레시피가 올라오게 만드는 강력한 원동력이 됩니다.

## 5. 🏷️ 해시태그 및 검색창 활성화
1. **인라인 해시태그**
   - 이제 본문을 작성할 때 `#올리브유`, `#바질`처럼 입력하면 앱이 자동으로 해시태그를 감지하여 파란색으로 칠합니다.
   - 이 파란색 태그를 누르거나 하단 검색 메뉴를 이용하면, 실시간으로 해당 키워드와 관련된 게시글들을 모아볼 수 있습니다.

## 6. 📱 스마트폰 앱(Android/iOS) 출시하기 (Capacitor)
웹사이트를 코드 수정 없이 그대로 모바일 앱 파일(`.apk`, `.ipa`)로 패키징할 수 있는 도구입니다.

```bash
npm install @capacitor/core @capacitor/cli
npx cap init foodhouse com.myfood.app --web-dir=.
npm install @capacitor/android
npx cap add android
npx cap sync
```
이 명령어들을 실행하면 Android Studio 프로젝트가 자동 생성되어 플레이스토어에 바로 올릴 수 있는 앱 빌드가 완성됩니다.

## 7. 🔑 카카오 로그인 실제 연결법
1. [카카오 개발자 센터](https://developers.kakao.com) 가입 후 애플리케이션 등록.
2. **플랫폼** 메뉴에서 내 웹사이트 주소(`https://내아이디.github.io/food`)를 등록.
3. 카카오 로그인 메뉴를 **활성화(ON)** 상태로 변경.
4. `index.html`에 카카오 SDK `<script src="https://developers.kakao.com/sdk/js/kakao.js"></script>` 추가.
5. 게스트 로그인 대신 `Kakao.Auth.login` 함수를 호출하도록 코드를 교체합니다.
